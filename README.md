# Job Portal

A simple web application built using Django to display and manage job listings.

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

## Setup

git clone <your-repo-link>
cd Job-Portal
pip install -r requirements.txt
python manage.py runserver

Open http://127.0.0.1:8000/

## Features
- View job listings
- Admin can add, update, and delete jobs
